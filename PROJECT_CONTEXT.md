# PROJECT CONTEXT: KasQ (Hybrid Offline-First POS)

## 1. System Overview
KasQ adalah transformasi dari aplikasi POS dan pembukuan keuangan UMKM berbasis web menjadi aplikasi mobile Android native (.APK) yang tangguh menggunakan **CapacitorJS**. Aplikasi mengadopsi prinsip **Offline-First**, di mana seluruh UI, mesin parser, dan database tersimpan lokal di dalam HP, namun memiliki kemampuan **Hybrid AI** yang secara adaptif memanfaatkan Google Gemini API ketika mendeteksi adanya koneksi internet untuk pemrosesan tingkat lanjut.

## 2. Tech Stack & Engineering Standards
* **Frontend Framework:** React.js / Vite / Tailwind CSS (Eksisting KasQ).
* **Mobile Wrapper:** CapacitorJS (`@capacitor/android` v6.x)[cite: 1].
* **Data Persistence (Offline DB):** IndexedDB (dioptimalkan menggunakan **Dexie.js** untuk query cepat, aman dari pembersihan otomatis OS)[cite: 1].
* **AI Orchestration Engine:**
    * *Online Engine:* Google Gemini API (`@google/generative-ai`) dengan penerapan *Structured Outputs* guna memaksa hasil dalam format skema JSON yang konsisten[cite: 1].
    * *Offline Engine:* Native Android Speech Recognition API (via `@capacitor-community/speech-recognition`) digabungkan dengan fungsi *Custom Regex & Rule-Based NLP Parser* lokal[cite: 1].
* **CI/CD Pipeline:** GitHub Actions runner (`ubuntu-latest`) dikonfigurasi bersama Java JDK 17 dan Android SDK Command-line Tools untuk otomatisasi kompilasi build `.apk`[cite: 1].
